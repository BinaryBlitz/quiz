## README

Steps to get the application up and running.

1. Check out
2. Run `$ bundle install`
3. Set up database at `config/database.yml`
4. Run `$ rake db:setup`
5. Run `$ rails server`

## Ruby version
Ruby 2.2.2

## How to run the test suite
Run `$ rake test`

## Deployment instructions
Run `$ bundle exec cap deploy`
